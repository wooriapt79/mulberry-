# AP2 Agent Payment Protocol v0.1

Date: 2026-03-06

## Overview

AP2 (Agent Payment Protocol) enables secure payment execution between
autonomous AI agents.

## Payment Principles

-   Identity-first authorization
-   Autonomous transactions
-   Interoperable payment infrastructure

## Core Components

-   Agent Passport Verification
-   Payment Authorization
-   Transaction Execution
-   Settlement Ledger

## Payment Flow

Agent Action → Order Creation → Payment Request → Passport Verification
→ Transaction Execution → Settlement

## Payment Targets

-   Agent-to-Agent
-   Agent-to-Merchant
-   Cooperative purchase settlement

AP2 forms the **Payment Layer** in the Mulberry Protocol.
