## ✅ 1. Senior Storytelling Engine

**파일**: `app/agents/sns_manager.py` (+300 줄)

### A. 핵심 기능: `generate_senior_story()`

**입력 → 출력**:
```python
# 입력 (시니어의 투박한 키워드)
"배추 수확함, 날씨 좋음"

↓ Qwen AI 스토리텔링 ↓

# 출력 (3가지 감성적 버전)
버전 A (따뜻한 일상):
"오늘 아침, 푸른골농원에서 싱싱한 배추를 수확했습니다. 
맑은 하늘 아래 정성껏 키운 배추가 드디어 제 모습을 드러냈습니다. 
인제군의 깨끗한 공기와 물로 자란 건강한 배추입니다. 🥬"

버전 B (계절 감성):
"겨울 인제의 차가운 공기가 배추를 더욱 아삭하게 만들었습니다. 
오늘 같은 맑은 날, 수확의 기쁨이 배가 됩니다. 
자연이 주는 선물에 감사하며 정성껏 수확했습니다. ❄️🌿"

버전 C (정성과 자부심):
"푸른골농원 김철수입니다. 
오늘 정성껏 키운 배추를 수확했습니다. 
인제군의 청정 자연에서 자란 우리 배추는 특별합니다. 
늘 건강한 먹거리로 보답하겠습니다. 💚"

# 자동 해시태그
#인제군 #로컬푸드 #푸른골농원 #배추 #수확 #청정자연
```

### B. Photo-Context-Sync (사진 맥락 연동)

**사진 메타데이터 활용**:
```python
photo_metadata = {
    "date": "2024-02-12",
    "location": "인제군 기린면",
    "weather": "맑음",
    "temperature": "5°C"
}

↓

"오늘(2024년 2월 12일) 인제군 기린면의 맑은 하늘 아래..."
```

**날씨 API 연동** (향후):
- OpenWeatherMap API
- 실시간 기온, 날씨 정보
- 자동 서두 작성

### C. 마스토돈 규격 포맷팅

```python
# 자동 포맷팅
- 최대 500자 제한
- 해시태그 자동 추가
- 이모지 2-3개 포함
- 줄바꿈 최적화
```

### D. 실전 사용 플로우

```
시니어 입력: "사과 포장 중"
  ↓
Qwen AI 스토리텔링 (3가지 버전)
  ↓
시니어가 마음에 드는 버전 선택
  ↓
마스토돈 자동 포스팅
  ↓
Strategy Advisor에게 활동 보고
```

### D. Photo-Context-Sync 구현

```javascript
// 사진 업로드 → 메타데이터 추출
photoInput.addEventListener('change', function(e) {
    const file = e.target.files[0];
    
    // EXIF 데이터 추출
    extractPhotoMetadata(file);
    
    // 날씨 API 호출
    getWeatherData().then(weather => {
        photoMetadata = {
            date: "2024-02-12",
            location: "인제군 기린면",
            weather: weather.condition,
            temperature: weather.temperature
        };
    });
});

"오늘(2024년 2월 12일) 인제군 기린면의 맑은 하늘 아래..."
```

## ✅ 3. 관제 시스템(Sentinel) 데이터 시각화 보강

**파일**: `app/agents/strategy_advisor.py` (+150 줄)

### A. 새로운 SNS 활동 지표

```python
sns_metrics = {
    "total_posts_today": 25,           # 오늘 포스트 수

        "#로컬푸드", 
        "#배추"
    ],
    "sentiment_score": 0.85            # 감성 점수 (긍정 85%)
}
```

### C. 실시간 추적 플로우

```

  ↓

- total_posts_today += 1
- senior_posts_today += 1
- reach += 100
  ↓
Sentinel 대시보드 자동 반영

Malu가 실시간 모니터링 📊
```
