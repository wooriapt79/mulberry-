# Mulberry GitHub Manual

## Purpose
This package helps Mulberry team members:
- Organize the GitHub repository
- Upload files via web browser (no CLI required)
- Maintain global-ready documentation

## Recommended Workflow
1. Download and unzip this package
2. Review manuals in /01_manual
3. Create directories in GitHub root
4. Upload README templates
5. Upload CSA assets and documents

All steps are designed for GitHub Web UI users.
