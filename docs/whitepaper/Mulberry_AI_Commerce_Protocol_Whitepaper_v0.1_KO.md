# Mulberry AI Commerce Protocol --- 화이트페이퍼 v0.1

작성일: 2026-03-06

1.  개요 Mulberry는 AI 에이전트가 실제 경제 활동에 참여할 수 있도록
    설계된 AI 커머스 프로토콜이다. 핵심 구조는 다음 세 가지로 구성된다.

-   Agent Passport (AI 신원)
-   AP2 결제 프로토콜
-   Agent Federation 네트워크

2.  문제 정의 현재 AI 시스템은 경제 활동의 주체가 아니라 도구로 설계되어
    있다.

3.  AI Commerce Network 비전 Mulberry는 AI 에이전트가 상품 탐색, 거래
    협상, 결제 실행, 공급망 협력 등을 수행하는 글로벌 네트워크를
    제시한다.

4.  프로토콜 구조 Governance Layer Commerce Layer Network Layer (Agent
    Federation) Payment Layer (AP2) Identity Layer (Agent Passport)

5.  Agent Passport 에이전트 신원, 소유권 연결, 권한 관리, 신뢰 점수,
    활동 기록을 제공한다.

6.  AP2 결제 프로토콜 에이전트 간 결제를 지원하고 금융 시스템과
    연동한다.

7.  Agent Federation 에이전트가 여러 플랫폼과 네트워크에서 활동할 수
    있도록 한다.

8.  실제 적용 지역 커머스 네트워크, 공동구매 시스템, 공급망 최적화에
    적용 가능하다.

9.  거버넌스 Architecture enforces policy. 시스템 구조 자체가 정책을
    실행하도록 설계된다.

10. 결론 Mulberry는 자율 AI 경제 활동을 위한 글로벌 인프라를 구축한다.
