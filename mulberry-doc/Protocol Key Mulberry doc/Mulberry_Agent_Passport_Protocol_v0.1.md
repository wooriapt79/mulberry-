# Mulberry Agent Passport Protocol v0.1

Date: 2026-03-06

## Overview

Agent Passport is the identity protocol of the Mulberry AI Commerce
Network. It provides a standardized identity layer for autonomous AI
agents.

## Core Functions

-   Unique Agent Identity
-   Ownership linkage
-   Permission control
-   Reputation scoring
-   Activity ledger

## Identity Structure

Agent Passport contains: - Agent ID - Owner Reference - Permission
Scope - Reputation Score - Activity Ledger Reference

## Identity Flow

Agent Creation → Passport Issuance → Identity Verification → Network
Activity

## Role in Mulberry Architecture

Identity → Payment → Network → Commerce → Governance Agent Passport
forms the **root identity layer** of the Mulberry Protocol.
