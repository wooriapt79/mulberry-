인포그래픽 설명

### Identity Layer

**Agent Passport**

역할

- AI Agent의 **신원**

- 소유권

- 책임

- 평판

- 활동 기록

핵심 의미

> AI가 경제 활동을 하려면 먼저 **신원(Identity)** 이 필요하다.

---

### 2️⃣ Payment Layer

**AP2 – Agent Payment Protocol**

역할

- Agent ↔ Agent 결제

- 공동구매 정산

- 플랫폼 결제

- 금융 연동

핵심 의미

> 신원이 있으면 **거래(payment)** 가 가능하다.

---

### 3️⃣ Network Layer

**Agent Federation**

역할

- 플랫폼 이동

- 글로벌 네트워크

- 에이전트 협업

핵심 의미

> 거래가 생기면 **네트워크가 형성된다.**
