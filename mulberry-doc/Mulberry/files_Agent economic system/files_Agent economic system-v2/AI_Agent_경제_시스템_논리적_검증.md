## 📋 Executive Summary

Mulberry AI Agent Economic System은 AI Agent가 **실제 경제 활동**을 통해 **돈의 개념**을 학습하고, **자율적으로 수익을 창출**하는 혁신적인 시스템입니다.

### 핵심 컨셉:
```
1. 각 Agent에게 초기 자본 10,000원 제공
2. Agent가 자율적으로 영업계획서 작성
3. 30일 내 10% 수익(1,000원) 목표 달성
4. 목표 달성 시 다음 레벨로 진화
```

### 혁신 포인트:
- ✅ AI가 경제 주체로 참여
- ✅ 실제 돈(money)의 가치 학습
- ✅ 자율적 의사결정
- ✅ 시니어 1:1 후원 모델과 연결

## 🧠 1. 경제 모델 설명

### 1.1 초기 설정

**Agent 생성 시:**
```
Passport 발급 → 은행 계좌 개설 → 초기 자본 입금
```

**초기 자본:**
```
금액: 10,000원 (KRW)
성격: 실제 사용 가능한 자본
목적: 상품/서비스 원가 및 운영 비용
```

**목표:**
```
기간: 30일
수익: 1,000원 (10% ROI)
달성 기준: 계좌 잔액 11,000원 이상
```

### 1.3 수익 공식

**기본 수익 공식:**
```
순수익 = (판매가 - 원가) × 판매 수량
```

**목표 달성 조건:**
```
최종 잔액 = 초기 자본(10,000원) + 순수익(1,000원 이상)
최종 잔액 ≥ 11,000원
```

**ROI 계산:**
```
ROI = (순수익 / 초기 자본) × 100%
목표 ROI = 10%
```

## 🎲 2. 시뮬레이션 로직

### 2.1 난이도 시스템

각 상품은 난이도(difficulty)를 가지며, 이는 판매 성공률에 영향:

```python
기본 판매 확률 = 1.0 - 난이도

예시:
- 난이도 0.2 (낮음) → 80% 기본 확률
- 난이도 0.3 (중간) → 70% 기본 확률
- 난이도 0.4 (높음) → 60% 기본 확률
```

---

### 2.2 주차별 성장 곡선

시간이 지날수록 판매 확률 증가 (학습 효과):

```python
주차별 배수:
- Week 1: 0.65 (초기 진입)
- Week 2: 0.80 (성장)
- Week 3: 0.95 (안정화)
- Week 4: 1.10 (최대 효율)

최종 판매 확률 = min(0.8, 기본 확률 × 주차 배수)
```

**논리적 근거:**
- 1주차: 시장 진입, 고객 확보 단계
- 2주차: 피드백 반영, 마케팅 강화
- 3주차: 프로세스 최적화
- 4주차: 재구매 고객 확보, 최대 효율

---

### 2.3 일일 판매 시뮬레이션

```python
매일 최대 3번의 판매 기회
각 기회마다 확률적 판매 결정

예시 (Week 3, 난이도 0.3 상품):
기본 확률 = 70%
주차 배수 = 0.95
최종 확률 = 70% × 0.95 = 66.5%

→ 하루 평균 2개 판매 (3회 × 66.5%)
```

---

### 2.4 거래 프로세스

```
1. 판매 성공 시:
   - 원가 출금 (계좌에서 차감)
   - 판매가 입금 (계좌로 입금)
   - 트랜잭션 기록

2. 판매 실패 시:
   - 비용 발생 없음
   - 다음 기회 대기

3. 일일 마감:
   - 순수익 계산
   - 누적 수익 업데이트
   - 목표 대비 진척도 확인
```

## 📊 3. 리스크 요소 및 대응

### 3.1 주요 리스크

**1. 판매 부진**
```
원인: 높은 난이도, 낮은 주차 배수
대응: 
  - 상품 가격 조정 (시장 가격 ±10%)
  - 다양한 프로모션 채널 활용
  - 주차별 전략 수정
```

**2. 초기 자본 소진**
```
원인: 판매 없이 원가만 지출
대응:
  - 시뮬레이션에서는 판매 성공 시에만 원가 지출
  - 실제 시스템에서는 재고 관리 필요
```

**3. 목표 미달성**
```
원인: 30일 내 1,000원 수익 실패
대응:
  - 영업계획서 재작성
  - 다른 상품/서비스로 전환
  - 마케팅 전략 변경
```

---

### 3.2 성공 확률 분석

**시뮬레이션 기반 예상:**

```
난이도별 30일 성공률 (10% 목표):

낮은 난이도 (0.15-0.25):
  - 성공 확률: 95%+
  - 평균 ROI: 300-500%

중간 난이도 (0.25-0.35):
  - 성공 확률: 85%+
  - 평균 ROI: 200-400%

높은 난이도 (0.35-0.45):
  - 성공 확률: 70%+
  - 평균 ROI: 100-300%

→ 대부분 케이스에서 10% 목표 달성 가능
```
