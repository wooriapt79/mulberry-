## 📊 3가지 시나리오 비교

### 성과 비교표

| Agent | 상품 | 난이도 | 최종 수익 | ROI | 결과 |
|-------|------|--------|-----------|-----|------|
| 김사과 | 시니어 지원 | 0.3 | 145,544원 | 1,455% | ✅ 성공 |
| 이번역 | 문서 번역 | 0.2 | 504,900원 | 5,049% | ✅ 초대성공 |
| 박감자 | 감자→데이터분석 | 0.35→0.3 | 420,750원 | 478% (2차) | ✅ 재도전 성공 |

---

### 핵심 인사이트

**1. 난이도가 성공의 핵심**
```
낮은 난이도 (0.2) → 초대성공 (5,049% ROI)
중간 난이도 (0.3) → 안정적 성공 (1,455% ROI)
높은 난이도 (0.35) → 변동성 큼
```

**2. 마진율의 중요성**
```
고마진 (90%): 문서 번역 → 최고 수익
중마진 (65%): 시니어 지원 → 안정적
저마진 (39%): 감자 → 변동성 큼
```

**3. 상품 선택 전략**
```
추천: 지식 서비스, 디지털 상품
주의: 농산물 (변동성 관리 필요)
```

**4. 실패도 성장의 과정**
```
박감자 Agent:
- 1차 실패 → 전략 수정 → 2차 대성공
- AI의 학습과 진화 증명
```
