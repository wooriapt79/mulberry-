## 📊 핵심 기능

### 1. Mastodon + ActivityPub 통합

**OAuth 인증:**
```python
# Mastodon 계정으로 로그인
oauth = MastodonOAuthIntegration(
    instance_url="https://mastodon.social",
    client_id="...",
    client_secret="..."
)

# 인증 URL 생성
auth_url = oauth.get_authorization_url(redirect_uri)

# 토큰 교환
access_token = oauth.exchange_code_for_token(code, redirect_uri)
```

**타임라인 자동 포스팅:**
```python
# 공동구매 시작 시 자동 포스팅
campaign = manager.create_campaign(product_id)
# → Mastodon 타임라인에 자동 공유

# 진행 상황 업데이트 (5명씩 참여할 때마다)
manager.join_campaign(campaign_id, user_id)
# → 자동으로 진행률 업데이트 포스팅
```

**ActivityPub Object:**
```python
# 연합 네트워크에 공유
product.to_activitypub_object()

# }
```

### 2. 공동구매 시스템

**상품 등록:**
```python
product = manager.create_product(
    name="인제 옥수수 1박스",
    description="강원도 인제군에서 자란 신선한 옥수수",
    category=ProductCategory.AGRICULTURAL,
    producer_agent_id="AGENT-INJE-001",
    producer_location="강원도 인제군",  # 식품사막화 지역
    original_price=30000,
    group_price=20000,  # 33% 할인
    min_quantity=20
)
```

**캠페인 시작:**
```python
campaign = manager.create_campaign(
    product_id=product.product_id,
    duration_days=7
)
# → 자동으로 Mastodon 타임라인에 포스팅
```

**참여 처리:**
```python
result = manager.join_campaign(
    campaign_id=campaign.campaign_id,
    user_id="user@mastodon.social",
    quantity=2
)
# {
#   "success": True,
#   "order_id": "ORDER-...",
#   "campaign": {
#     "current_participants": 15,
#     "progress_percent": 75.0
#   }
# }
```

**핫딜 조회:**
```python
hot_deals = manager.get_hot_deals(10)
# 현재 진행중인 인기 공동구매 목록
```

### 3. 식품사막화 → 도시 연결

**우리 마을 공동구매:**
```python
# 인제군 생산품 공동구매
purchases = manager.get_village_purchases("강원도 인제군")
# → 서울, 춘천 등 도시 소비자 참여
# → 공동 배송으로 물류비 1/20 절감
```

**C2M (Consumer to Manufacturer) 모델:**
```
농가 → [중간 유통 제거] → 소비자
✅ 농가: 도매가 대신 소비자가 (30% ↑)
✅ 소비자: 신선한 상품 저렴하게
✅ 공동구매로 배송비 절감
```
