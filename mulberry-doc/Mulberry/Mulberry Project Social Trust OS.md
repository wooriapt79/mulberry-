# 🌳 Mulberry Project: Social Trust OS

> **"기술의 차가움에 인간의 온기를 더하다. 0.70에서 시작하는 새로운 신뢰의 로직"**

## 🚀 Project Overview

Mulberry Project는 AI 기술과 인간의 따뜻한 연결을 결합한 차세대 **Social Trust Operating System**입니다. 우리는 단순히 효율적인 기술을 만드는 것이 아니라, 디지털 소외 계층을 포함한 모든 이들이 기술의 혜택을 누릴 수 있도록 돕는 **'장승배기 로직(Jang-seung-baegi Logic)'**을 구축합니다.

## ⚖️ Core Concept: Spirit Score (0.70)

모든 팀원과 AI는 **Spirit Score 0.70**이라는 공평한 지점에서 시작합니다. 

- **Human-in-the-loop (HITL):** 모든 기술적 결정에는 인간의 맥락, 온기, 그리고 윤리적 판단이 개입됩니다.
- **Dynamic Ledger:** 신뢰와 기여도는 실시간으로 측정되며, 이는 Mulberry 생태계의 핵심 지표가 됩니다.

## 🛠️ Tech Stack & Infrastructure

- **Core Engine:** Jang-seung-baegi Logic v2.1 (Based on Gemini & Vertex AI)
- **Edge Computing:** Raspberry Pi 5 based decentralized nodes
- **Communication:** Slack Integrated Bot (Malu & Koda)
- **Infrastructure:** Google Cloud Platform (GCP) Enterprise Environment
