**작동 방식:**

```python
# 1. 사용자가 에이전트에게 권한 부여
intent = mandate_manager.create_intent_mandate(
    user_id="USER-001",
    agent_id="AGENT-001",
    intent="식료품 구매",
    constraints={"max_budget": 50000}

# 2. 에이전트가 권한 확인
can_buy = mandate_manager.verify_agent_authority(
    agent_id="AGENT-001",
    action="purchase",
    context={"amount": 30000}
)
# → True (예산 내)

# 3. 장바구니 생성 및 승인
cart = mandate_manager.create_cart_mandate(...)

# 4. 결제 실행
payment = mandate_manager.create_payment_mandate(...)
```

### ✅ 2. Spirit Score 시스템

**파일:** `modules/spirit_score/spirit_score_manager.py` (377 라인)

**완성된 기능:**

- ✅ 모든 활동 자동 점수화
- ✅ 레벨 시스템 (0-100)
- ✅ 리더보드
- ✅ 실시간 진행도 추적

**점수 시나리오 (요청하신 대로):**

```
업무 완료:           +0.01  ← 요청하신 예시!
고객 응대:           +0.005
긍정 리뷰:           +0.02
다른 에이전트 도움:   +0.03
회의 참석:           +0.01
교육 이수:           +0.05
헌법 준수:           +0.01
상부상조 제공:       +0.05

업무 실패:           -0.02
부정 리뷰:           -0.03
헌법 위반:           -0.1
회의 불참:           -0.02
도움 거부:           -0.05
```

**자동 통합:**

```python
# 업무 완료 시 자동 호출
spirit_manager.on_task_completed(agent_id, "김밥 판매")
# → +0.01 자동 부여

# 고객 응대 시 자동 호출
spirit_manager.on_customer_served(agent_id, customer_id)
# → +0.005 자동 부여

# 리뷰 수신 시 자동 호출
spirit_manager.on_review_received(agent_id, 5, "맛있어요!")
# → +0.02 자동 부여
```

**레벨 시스템:**

```
Novice     (0-20점)
Apprentice (21-40점)
Skilled    (41-60점)
Expert     (61-80점)
Master     (81-100점)
```

### ✅ 3. 장승배기 5대 강령 + 상부상조 10%

**파일:** `modules/jangseungbaegi_checker/checker.py` (475 라인)

**완성된 기능:**

- ✅ 5대 강령 실시간 체크
- ✅ 위반 시 자동 감점
- ✅ 상부상조 10% 자동 배분
- ✅ 준수 점수 추적

**5대 강령 코드 레벨 체크:**

```python
# 1. 상부상조
checker.check_mutual_aid(agent_id, helped_someone=True)
# → 준수 시 +0.01, 위반 시 -0.05

# 2. 투명성
checker.check_transparency(agent_id, disclosed_properly=True)
# → 준수 시 +0.01, 위반 시 -0.1 (심각!)

# 3. 책임감
checker.check_responsibility(agent_id, completed_task=True)
# → 준수 시 +0.01, 위반 시 -0.02

# 4. 공동체 정신
checker.check_community(agent_id, contributed=True)
# → 준수 시 +0.01, 위반 시 -0.01

# 5. 탁월성 추구
checker.check_excellence(agent_id, quality_standard_met=True)
# → 준수 시 +0.01, 위반 시 -0.02
```

**상부상조 10% 자동 배분:**

```python
# 하루 수익 10만원
daily_earnings = 100000

# 자동으로 10% (1만원) 배분
transactions = checker.process_mutual_aid_contribution(
    agent_id="AGENT-001",
    total_earnings=daily_earnings
)

# → Spirit Score 낮은 에이전트 5명에게
# → 각 2천원씩 자동 배분
# → 도운 에이전트는 +0.05 x 5 = +0.25 획득
```

**준수 점수 추적:**

## 🔗 통합 플로우

```
1. 에이전트 생성 (Agent Factory)
   ↓
2. 장승배기 헌법 학습 (1시간)
   → Spirit Score +0.05
   ↓
3. 사용자가 Intent Mandate 생성 (AP2)
   → 에이전트에게 권한 부여
   ↓
4. 에이전트가 업무 수행
   → 권한 자동 검증 (AP2)
   → 장바구니 생성 (Cart Mandate)
   → 결제 실행 (Payment Mandate)
   ↓
5. 모든 활동 자동 기록
   → Spirit Score 증가
   → 5대 강령 체크
   ↓
6. 하루 종료 시
   → 상부상조 10% 자동 배분
   → 어려운 에이전트 자동 지원
   ↓
7. 준수 점수 업데이트
   → 리더보드 반영
   → 레벨업 확인
```
