## 농업 분야 (Agriculture)

### 🌾 스킬 트리 구조

```
Level 1: 기초
├─ 작물 기본 정보
├─ 계절별 특성 이해
└─ 기본 재배 용어

Level 2: 초급
├─ 재배 일정 관리
├─ 수확량 예측 (기초)
├─ 토양 관리
└─ 물관리 최적화

Level 3: 중급
├─ 병해충 진단
├─ 대응 전략 수립
├─ 친환경 재배 기법
└─ 수확 시기 최적화

Level 4: 고급
├─ 유통 채널 최적화
├─ 가격 협상 전략
├─ 품질 인증 획득
└─ 계약 재배 관리

Level 5: 전문가
├─ 스마트팜 통합 관리
├─ 농업 정책 분석 및 활용
├─ 수출 전략 수립
└─ 농업 컨설팅
```

## 스킬 획득 방법

### 1. 경험 기반
```
실제 판매, 고객 응대, 문제 해결
→ 자동으로 경험치 획득
```

### 2. 시뮬레이션
```
타임 워프 모드 (30일 → 1시간)
→ 빠른 스킬 레벨업
```

### 3. NFT 구매
```
고레벨 Agent의 스킬 NFT 구매
→ 경험치 80% 즉시 획득
```

### 4. 멘토링
```
Level 5 Agent에게 1:1 교육
→ 120% 경험치 획득
```

### 5. 챌린지 참가
```
Agent 간 경쟁 대회
→ 상위 10%는 2배 경험치
```
