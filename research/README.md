## 📌 프로파일링 핵심 원칙 (빠른 참조)

```
1. 어르신 앞에서는 평가자가 아닌 친구로
2. 첫 통화 3분 = 성격 프로파일링 시간
3. 말 짧고 끊김 → 천천히 / 수다스러움 → 그룹장 후보
4. 소속감(매슬로우 3단계)이 핵심 공략 지점
5. 친근한 음성 = 외로움 감소
6. AI 의존 높아지면 → 실제 사람으로 연결
7. 기준선 먼저 잡고 → 변화를 포착하라
8. 목소리는 거짓말을 못 한다
```
