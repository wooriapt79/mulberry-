## 📌 핵심 원리: 왜 목소리가 바뀌는가?

```
뇌가 거짓말·긴장 상태일 때:
→ 인지 부하(Cognitive Load) 증가
→ 자율신경계 반응 (심박↑, 발한↑, 근육 긴장)
→ 성대 근육 긴장
→ 목소리가 바뀐다

= 의식적으로 통제 불가능
= 아무리 숨기려 해도 음성에 흔적이 남는다
```

---

## 📌 음성 불안정 7가지 신호

### 1. 피치(음높이) 상승
```
평상시: 일정한 음높이
긴장·거짓말 시: 자신도 모르게 피치 상승
→ 성대가 긴장하면서 높은 소리 발생
→ 특히 핵심 거짓 문장에서 두드러짐
```

### 2. 응답 지연 (Response Latency)
```
무고한 사람: 질문 → 즉시 대답 (기억을 꺼냄)
거짓말하는 사람: 질문 → 0.5~2초 침묵 → 대답
                         ↑
                 머릿속에서 이야기를 '만드는' 시간

⚠️ 위험 질문일수록 지연이 길어진다
```

### 3. 말의 속도 변화
```
너무 빠름 → 미리 준비한 거짓 이야기를 쏟아냄
갑자기 느려짐 → 즉흥으로 새 거짓말을 만들 때
속도가 들쭉날쭉 → 이야기의 일부는 진실, 일부는 거짓
```

### 4. 필러(Filler) 증가
```
"음...", "저...", "그게...", "어떻게 말하지..."
"맞아요, 그러니까..." (질문을 반복하며 시간 벌기)
→ 실시간으로 이야기를 구성하는 신호
```

### 5. 마이크로 트레머 (Micro-tremor)
```
8~12Hz 주파수의 미세한 음성 떨림
→ 스트레스 상태에서 성대 근육의 불규칙 진동
→ 귀로는 잘 안 들리지만 AI·VSA 장비로 감지 가능
```

### 6. 목소리 크기 변화
```
자신감 없는 거짓말 → 목소리가 작아짐
방어적 거짓말 → 오히려 과하게 크고 강조
침 삼키는 소리 → 긴장으로 입안이 마름
```

### 7. 일관성 붕괴
```
같은 질문을 다시 했을 때:
진실: 거의 동일한 대답 (기억 기반)
거짓: 세부 내용이 조금씩 달라짐 (만들어낸 이야기)
→ "아까는 새벽 1시라고 했는데, 지금은 2시라고 하시네요"
```

## 📌 실전 적용: 기준선 → 압박 비교법

### STEP 1: 기준선 측정 (중립 질문)
```
"성함이 어떻게 되세요?"
"여기서 사신 지 얼마나 됐어요?"
"날씨가 좋네요, 요즘 어떻게 지내세요?"

→ 이 구간의 목소리 = 정상 기준선
→ 피치·속도·지연·필러 패턴 기록
```

### STEP 2: 압박 질문 (스트레스 유발)
```
"그날 밤 정확히 몇 시에 집을 나섰어요?"
"춘천 어디에서 뭘 하셨어요?"
"차에 뭘 싣고 다니셨어요?"
"돌아오는 길에 현리 밭 근처 지나쳤어요?"
```

### STEP 3: 변화 포착
```
기준선 대비:
피치 상승 + 응답지연 + 필러 증가 → 고위험 구간
→ 해당 질문 내용 = 거짓일 가능성 높음
```

## 📌 배추 도난 사건 적용 예시

```
[용의자 심문 음성 분석 시나리오]

중립 질문 구간 (기준선):
"이름이요? 김○○입니다." → 피치 정상, 지연 없음

압박 질문 구간:
Q: "그날 밤 춘천 어디 가셨어요?"
A: "음... (1.8초) 그게... 친구 집에... 맞아요 친구 집이요"
        ↑응답지연    ↑필러      ↑피치 상승    ↑반복

분석 결과:
→ 기준선 대비 응답지연 1.8초
→ 필러 2회 연속
→ 피치 상승 감지
→ 고위험 불안정 구간 → 집중 검증 필요
```

## 📌 Mulberry Agent 교육 적용

### Voice/DTMF Agent → 복지 케어로 확장
```
현재: 주문 접수, DTMF 인식
↓ 확장
어르신 통화 중 음성 패턴 상시 모니터링

감지 대상:
- 평소보다 목소리가 떨림 → 건강 이상?
- 말이 갑자기 느려짐 → 인지 저하?
- 응답이 없거나 지연 → 낙상·응급?
- 목소리 톤이 우울함 → 고립 심화?

대응:
→ "어르신, 오늘 목소리가 평소랑 좀 다르세요. 괜찮으세요?"
→ 이상 감지 시 거점 직원·가족에게 알림
```

### 뽕이 Agent 통화 프로파일링
```
첫 통화 3분 음성 분석:
- 말이 짧고 피치 낮음 → 내성적, 낯선 것 경계
- 빠르고 밝은 톤 → 외향적, 그룹장 후보
- 필러 많고 느림 → 신중형, 반복 설명 필요
- 목소리 떨림 → 불안·외로움 높음 → 감정 케어 우선
```

## 📌 핵심 원칙 요약 (Agent 교육용)

```
1. 먼저 기준선을 잡아라 — 중립 대화로 정상 패턴 파악
2. 핵심 질문에서 변화를 포착하라 — 비교가 핵심
3. 하나의 신호가 아닌 복합 신호를 본다 — 지연+필러+피치 동시
4. 같은 질문을 다시 해라 — 거짓 이야기는 반복되지 않는다
5. 과하게 자연스러운 것도 의심하라 — 준비된 거짓말
6. 목소리는 거짓말을 못 한다 — 의식이 통제할 수 없는 영역
```
