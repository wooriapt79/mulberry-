## 핵심 개념: Observed Exposure (관찰된 노출도)

> AI가 특정 직업의 업무를 **실제로 얼마나 대체·보조하고 있는가**를 측정하는 새로운 지표

- 기존 연구의 한계: "이론적 가능성" 측정 (AI가 할 수 있는지)
- Observed Exposure: **실제 Claude 사용 데이터** 기반으로 직업별 AI 침투율 측정
- O*NET 800개+ 직업 분류 체계 적용

---

## 논문의 주요 발견

### AI 노출도 높은 직업군 (AI 빠른 침투)
- 소프트웨어 개발자, 데이터 분석가
- 법률 문서 작성, 금융 분석
- 콘텐츠 크리에이터, 번역가
- 의료 코딩, 행정 비서

### AI 노출도 낮은 직업군 (AI 미진입 = 블루오션)
- **농촌 현장 서비스** — 신체 노동 + 지역 신뢰 기반
- **복지 돌봄 서비스** — 감정 연결, 어르신 대화
- **로컬 음식 유통** — 하이퍼로컬, 방언·문화 맥락
- **공동구매 조직화** — 커뮤니티 관계 자본
- **전화 주문 접수** — DTMF, 사투리 대응

## Mulberry 관점: 왜 이 논문이 중요한가?

### 포지셔닝 확인
Anthropic 논문이 입증한 "AI 미진입 블루오션" = **Mulberry가 이미 진입한 시장**

| 논문의 저노출 직업 | Mulberry Agent 매핑 |
|---|---|
| 농촌 복지 동반자 | 뽕이 Agent (@bbongyi) |
| 전화 주문 접수 | Voice/DTMF Agent |
| SNS 커뮤니티 빌더 | Mastodon Marketer Agent |
| 콘텐츠 저널리스트 | Blog Writer Agent |
| 신원·신뢰 관리 | Passport Agent (DID) |

### 경쟁 우위
- 빅테크는 "고노출 직업"에 집중 (개발자, 분석가)
- Mulberry는 "저노출 블루오션"에 집중 → **경쟁자 없음**
- 인제군 어르신 = AI가 진입 못한 마지막 프런티어

## 관련 개념 메모

- **O*NET**: 미국 직업정보망, 800개+ 직업을 기술·능력·업무 단위로 분류
- **Observed vs Theoretical Exposure**: 이론적 가능성 vs 실제 사용 데이터 차이
- AI가 "대체"보다 "보조"로 작동하는 영역 → Mulberry의 Human+AI 모델과 일치
- 방언·감성·신뢰 = 정량화 어려운 요소 → AI 단독 진입 장벽 = Mulberry 해자(moat)
