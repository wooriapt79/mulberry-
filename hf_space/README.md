# 🌿 Mulberry Social-Agentic Commerce Demo

**식품사막화 제로 프로젝트** — Interactive Technology Demo for Google Engineers

## Overview

Mulberry is a Social-Agentic Commerce platform designed to eliminate food deserts in rural Korea.
This demo showcases three core technology pillars:

### 🏦 Payment Integration
- NH Nonghyup + AP2 real-time settlement
- Two-Phase Commit with ACID guarantees
- Automatic voucher denomination conversion
- Sub-200ms transaction latency

### 📡 Voice Protocol
- Offline voucher revocation via DTMF
- HMAC-SHA256 verification
- Works with 70%+ voice coverage (vs 99% data requirement)
- 13-second completion vs 30+ minutes (data-only)

### 🎮 Game Theory Market
- Spirit Score-based skill trading
- Nash Equilibrium optimization
- Dynamic price discovery
- Tournament league system

## How to Use

1. Select a tab (Payment / Voice / Market)
2. Adjust the input parameters
3. Click the action button
4. Explore the JSON result
